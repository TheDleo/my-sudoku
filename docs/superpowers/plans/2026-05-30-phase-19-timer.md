# Phase 19 — Timer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire a live elapsed-time counter that runs while the player is solving, displays in `GameHeader`, and stops permanently on win.

**Architecture:** A new `tickTimer` reducer increments `elapsedMs` by 1000ms. `GameHeader` drives a 1-second `setInterval` via `useEffect([won])` — stopping when `won` becomes true and auto-cleaning up when the component unmounts (on landing). `formatTime` is extracted from `WinModal` to `helpers.ts` so both components share it.

**Tech Stack:** React (`useEffect`, `setInterval`), Zustand, Vitest (`vi.useFakeTimers`), @testing-library/react (`act`)

```ts
export function formatTime(ms: number): string {
  const totalSeconds = Math.floor(ms / 1000);
  const minutes = Math.floor(totalSeconds / 60);
  const seconds = totalSeconds % 60;
  return `${minutes}:${seconds.toString().padStart(2, '0')}`;
}
```

Replace the full content of `src/game/GameHeader.tsx`:

```tsx
import './GameHeader.css';
import { useEffect } from 'react';
import type { Difficulty } from '../types';
import { formatTime } from './helpers';
import { useGameStore } from './store';

const DIFFICULTY_LABELS: Record<Difficulty, string> = {
  easy: 'Easy',
  medium: 'Medium',
  hard: 'Hard',
  expert: 'Expert',
};

export function GameHeader() {
  const difficulty = useGameStore((s) => s.puzzle.difficulty);
  const elapsedMs = useGameStore((s) => s.elapsedMs);
  const won = useGameStore((s) => s.won);

  useEffect(() => {
    if (won) return;
    const id = setInterval(() => useGameStore.getState().tickTimer(), 1000);
    return () => clearInterval(id);
  }, [won]);

  const handleNewGame = () => {
    const confirmed = window.confirm('Start a new game? Your current progress will be lost.');
    if (confirmed) {
      useGameStore.getState().setScreen('landing');
    }
  };

  return (
    <div className="game-header" onClick={(e) => e.stopPropagation()}>
      <span className="game-header__difficulty">{DIFFICULTY_LABELS[difficulty]}</span>
      <span className="game-header__time">{formatTime(elapsedMs)}</span>
      <button className="game-header__new-game" onClick={handleNewGame}>
        New Game
      </button>
    </div>
  );
}
```
